実行確認環境 windows10, 11
開発環境 python3.9


使用方法
1. displaychanger.zipを解凍します。
2. 中にあるdisplaychanger.exeをタスクバーにドラッグ&ドロップします。
3. タスクバーのdisplaychangerをクリックします。

フルスクリーンのゲーム中はAlt + Tabでタスクバーを表示して押すなどすると良いと思います。

気分が向いたら、特定のゲームをプレイしているときのみシングルディスプレイへ切り替えてフレームレートを向上させるアプリを作りたいと思います。
